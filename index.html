McInelly Modpack

How to Install the Modpack

  1. Download Twitch - Windows | Mac
  2. Click the download link below for the newest version
  3. Sign up for Twitch.tv using program after installing
  4. Click on Mods at the Top
  5. Select Minecraft
  6. Click Create Custom Profile
  7. At the top of the popup click import
  8. Navigate to and select the download from step 2
  9. Click play!
